# EMRE CRADA Manuscript

This repository develops a publishable journal manuscript on land-use change to perennial bioenergy crops in the Permian Basin, with process-modeling evidence from DAYCENT and life-cycle framing for greenhouse gas (GHG) outcomes.

## Current manuscript state

The active manuscript is `main.tex`. It contains a strong technical direction but is not submission-ready. The current draft still includes unfinished narrative fragments, open reviewer questions embedded as comments, placeholder citations, and sections that require tighter alignment between methods, results, and conclusions.

## Research focus and direction

Current analysis emphasizes net GHG balance across land-management scenarios. The near-term writing direction is to pivot the manuscript framing toward soil health, with soil organic carbon (SOC) as the primary organizing outcome. Net GHG remains in scope as a supporting systems metric rather than the lead story.

## Immediate documentation objective

Project documentation tracks how to move from the current draft to a submission-ready manuscript by:

1. Defining publishable scope and milestones.
2. Recording current blockers in the manuscript narrative.
3. Prioritizing SOC-centered analyses, interpretation, and figures.
